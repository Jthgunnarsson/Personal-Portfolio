Project 1 - The Website

Preperations

I started with watching the fyrst tutorial videos. To make the project a little bit easier i did exactly as the video instructed because i had never used HTML5 or made a website before.

I chose to use Brackets because that is the program that the teacher used.

Next I downloaded the template Editorial for the website on HTML5 UP and started setting up my website. I chose Editorial template because it has a sidebar i can costomize and i like the style.

Github

I started with signing up in github and seting up a new Repository under the name Personal-Portolio. To upload the files from Bracket to Github i simply dragged and dropped the files to the repository.

After that I went into "Settings" in the Github Repository and clicked the "Pages" tab. There I changed the Source from "none" to "main" and saved. With that my page is in online. :))

Changing the Website

I started with changing the code in index.html

There I made copies of generic.html file to Vel608G-Project1.html and Vel608G-Project2.html etc. for later projects. Then I changed the sidebar so it would have a subbar for this course and other courses im attending. After that I added a picture in Project 1.

Then I changed the homepage and added an introdutcion.

CV

I made a copy of Vel608G-Project1.html and saved it under the name CV.html. From there i started writing the fyrst part of my CV: introduction to me.
